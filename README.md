# Você está sendo codificado!
```
O grande problema de direitos humanos do nosso século são os agentes de inteligência artificial
construidos para perpetuar e automatizar diversas opressões sociais. Você está sendo codificado! 
é um video interativo para WEB e nele procuro abordar algumas violações de direitos humanos operadas 
por agentes. Um importante elemento da compasição é o espector que é inserido na diegese do vídeo
através da webcam. A camerá do espectador passa por filtros AR gerados por IA para explicar como
esses agentes funcionam e porque podem ser perigosos.
```

# Ferramentas de desenvolvimento
```
Você está sendo codificado! é um video interativo para WEB que utiliza p5.js, ml5.js e jquery.
```
